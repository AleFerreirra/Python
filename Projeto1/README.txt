# Análise de Dados de Vendas de E-commerce

## Objetivo
Analisar os dados de vendas de uma empresa de e-commerce para entender padrões de compra, identificar produtos mais vendidos, sazonalidades e insights acionáveis.

## Etapas do Projeto

1. **Coleta de Dados**: Carregamento do dataset de vendas.
2. **Pré-processamento dos Dados**: Limpeza e preparação dos dados para análise.
3. **Análise Exploratória dos Dados**:
    - Distribuição de preços dos produtos.
    - Vendas por categoria de produto.
    - Relação entre preço e avaliação dos produtos.

4. **Resumo dos Insights**:
    - A maioria dos produtos está em uma faixa de preço moderada.
    - Algumas categorias de produtos são significativamente mais populares.
    - Analisar a relação entre preço e avaliação ajuda a entender a percepção dos clientes.

## Ferramentas Utilizadas
- **Python**: Para manipulação e análise dos dados.
- **Pandas**: Para manipulação de dados.
- **Seaborn** e **Matplotlib**: Para visualização de dados.
- **VSCode**: Para documentação e execução do código.

## Como Reproduzir a Análise
1. Clone este repositório.
2. Instale as dependências necessárias.
3. Execute o notebook `analysis.ipynb` para reproduzir a análise e os gráficos.

## Conclusão
Esta análise fornece uma visão geral dos padrões de vendas de uma empresa de e-commerce, ajudando na tomada de decisões informadas sobre marketing, estoque e estratégia de preços.
